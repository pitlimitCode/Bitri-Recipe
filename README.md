### Format database yang ku buat:
| users table  | data type | not_null? | unique? | primary_key? | foreign_key |
| ------------ | --------- | --------- | ------- | -----------  | ----------- |
| id           | int       | y         |  y      | y            | -           |
| name         | varchar   | y         |  -      | -            | -           |
| email        | varchar   | y         |  -      | -            | -           |
| phone_number | int       | -         |  -      | -            | -           |
| password     | varchar   | -         |  -      | -            | -           |
| avatar       | varchar   | -         |  -      | -            | -           |

| recipes table | data type | not_null? | unique? | primary_key? | foreign_key |
| ------------- | --------- | --------- | ------- | ------------ | ----------- |

| ingredients   | varchar   | y         |  -      | -            | -           |
| step          | varchar   | -         |  -      | -            | -           |
| image         | varchar   | -         |  -      | -            | -           |
| video         | varchar   | -         |  -      | -            | -           |
